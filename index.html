// 02 — Projects

Hivecore
Python Local-First CLI Systems Design

A terminal-first personal assistant built for local, private use. It includes persistent memory, a managed home directory, run history, and explicit tool connectors. The main idea is simple: your assistant should run on your machine, under your control, with optional cloud models for better capabilities.

↗ GitHub
tcom
C Emulation Computer Architecture

An 8-bit computer emulator built from scratch, including the CPU, memory model, and instruction set. I built it to better understand computer architecture by implementing the moving parts myself.

↗ GitHub
HTTP Server
C++ POSIX Sockets Concurrency Networking

A lightweight HTTP/1.1 server built over raw POSIX sockets. It handles request parsing, static file serving, MIME detection, thread-per-connection concurrency, timeouts, request size limits, and directory traversal protection without relying on a framework.

↗ GitHub
Unix Shell
C POSIX Process Control

A minimal Unix shell in C with command parsing, external execution through fork/execvp, built-in commands, and signal handling that correctly targets child processes on interrupt.

↗ GitHub
TCP Port Scanner
C Winsock2 pthreads Security

A multithreaded TCP port scanner in C using Winsock2 and pthreads. It uses a queue-based worker model and supports banner grabbing, service detection, configurable timeouts, and per-scan timing stats.

↗ GitHub
Home Server
Ubuntu Server Linux Tailscale Self-Hosted

A self-hosted Ubuntu server with about 3TB of storage. I set up the OS, configured services, and handle ongoing administration myself, with remote access through Tailscale. It has mostly been a way to learn infrastructure by actually running it.
