# Personal Portfolio

A simple, responsive portfolio site showcasing my work, skills, and background. Built with HTML, CSS, JavaScript, and Bootstrap.

## Features
- Landing page with quick intro and links
- About page with background, interests, and skills
- Projects page with descriptions and GitHub links
- Contact page with email, GitHub, and LinkedIn
- Dark mode toggle
- Mobile-friendly layout

## Tech Stack
- HTML, CSS, JavaScript
- Bootstrap 5
- Bootstrap Icons
